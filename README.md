virtustan
=========

*Virtustan application*

~~эзотерическое~~ приложение, имеющее отношение
к виртуальному королевству Виртустан (www.virtustan.net)

По сути простейшая игра, некая смесь рогалика
([roguelike](http://lurkmore.to/%D0%A0%D0%BE%D0%B3%D0%B0%D0%BB%D0%B8%D0%BA%D0%B8))
и муда ([MUD](http://lurkmore.to/MUD))

В настоящее время реализованы команды:

Command list:
koi, win, utf, lat - switch codetable
codetable - print current codetable
alf - print alfavit
ascii - print ascii table
help, помощь - help
test - test
q, quit, exit, конец - exit

И самое главное - команды перемещения по сторонам света - n, s, w, e и команда "посмотреть" - look

И еще малая польза от данного приложения: им я проверяю, какая установлена кодировка кириллицы ;)

Приложение протестировано в средах Linux (Ubuntu, Debian, Centos), Windows 7 (в среде cygwin),
FreeBSD 9
